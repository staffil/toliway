# 🚇 TOILWAY

> 지하철 및 공공장소 화장실 위치 정보 제공 웹 서비스

---

## 📌 프로젝트 소개
서울 및 경기도 권의 수많은 직장인들을 위해 만들었습니다. 한국의 지하철 내부 화장실은 환경과 관리가 깔끔하게 되어있지만 그 위치를 파악하는데 큰 어려움이 있습니다. 또한 직장인들이 출근길에 급하게 용변을 처리해야 할 경우 안내 표지판을 보며 갈 수 있지만 위치가 헷갈릴떄도 있고 처음 내리는 역인데 자리가 없어서 헤맨 기억이 많으실 겁니다. 이럴 경우를 대비해 직장인들을 위한 지하철 공중 화장실 위치와 대변기수를 알려주는 사이트를 만들었습니다. 또한 현재 시간 기준으로 해당 역에서 승차, 하차 총 인원의 혼잡도 정보를 볼 수 있습니다.

## 🔗 사용 API

| API 명 | 제공처 | 용도 |
|--------|--------|------|
| 지하철역 화장실 POI | 서울 공공데이터포털 | 지하철역 화장실 상세 정보 조회 |
| 서울 공중화장실 POI | 서울 열린데이터광장 | 서울시 공중화장실 위치 및 시설 정보 |
| 경기도 공중화장실 | 경기도 공공데이터포털 | 경기도 내 공중화장실 정보 |
| 서울 지하철 시간대별 이용현황 | 서울 열린데이터광장 | 시간대별 승하차 인원 혼잡도 데이터 |
| 서울 스마트 비상구 안내 이미지 | 서울 열린데이터광장 | 지하철역 비상구 안내 이미지 제공 |

## 🚀 주요 기능

- **지하철 화장실 검색** — 역명으로 검색 시 남/여 화장실 수, 위치, 장애인 시설 정보 제공
- **지역 화장실 검색** — 서울·경기 지역의 건물·공공장소 화장실 정보 검색
- **지하철 혼잡도** — 시간대별 승하차 인원 기반 혼잡도 색상 표시 (6만 명 기준)
- **비상구 안내도** — 지하철역 비상구 이미지 조회
- **로그인** — 자체 로그인 및 Google/Naver/Kakao 소셜 로그인 UI
